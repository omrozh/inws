InStore

InStore is a console based file storage service developed by the InGames, Inc. You can store and share your files through single line terminal commands everywhere in the world.
Install InStore
MacOS & Linux

Open terminal

Enter the following commands:
curl https://storagerequests.herokuapp.com/client/install >>Â instore.zip
unzip ~/instore.zip
Set your startup command to "source ~/startshell.sh" for auto loading

Create account
instore createAccount yourusername yourpassword
Login to account
instore login yourusername yourpassword

You need to login to perform any operations.

Upload file
instore upload filepath
Download file
instore download filename savepath
Share file
instore addOwner sharedaccountusername filename
Delete file
instore delete filename
Search for a file
instore search filename
List all files
instore list
