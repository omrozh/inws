InWS

InWS(InWebStorage) is a console based file storage service developed by the InGames, Inc. You can store and share your files through single line terminal commands everywhere in the world.

Warning: Make sure not to use special characters and Non-English characters while naming or files or opening an account!
Install InWS
MacOS

Open terminal

Enter the following commands:
curl http://www.inws.ml/client/install >>Â inws.zip
unzip ~/inws.zip
Set your startup command to "source ~/inws.sh" for auto loading

Create account
inws createAccount yourusername yourpassword
Login to account
inws login yourusername yourpassword

You need to login to perform any operations.

Upload file
inws upload filepath
Download file
inws download "filename" savepath
Share file
inws addOwner sharedaccountusername "filename"
Delete file
inws delete "filename"
Search for a file
inws search "filename"
List all files
inws list
